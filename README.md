# Анализ успеваемости студентов

## Описание проекта

Цель проекта — исследовать факторы, связанные с результатами студентов на итоговом экзамене.

В рамках анализа рассматриваются:

- посещаемость занятий;
- время, затрачиваемое на учёбу;
- доступ к интернету;
- уровень образования родителей;
- наличие подработки;
- студенты с результатами выше среднего;
- потенциальная группа риска;
- лидеры внутри отдельных категорий.

Для анализа использовалась база данных PostgreSQL и набор SQL-запросов различного уровня сложности: агрегатные функции, группировка, подзапросы, CTE и оконные функции.

## Цели анализа

1. Получить общую характеристику набора данных.
2. Сравнить средний экзаменационный балл между различными группами студентов.
3. Определить факторы, сильнее связанные с итоговым результатом.
4. Найти студентов с результатами выше среднего.
5. Выделить потенциальную группу риска.
6. Продемонстрировать применение продвинутых SQL-конструкций.

## Структура набора данных

Таблица `students` содержит информацию о 1000 студентах.

Основные поля:

| Поле | Описание |
|------|----------|
| student_id | идентификатор студента |
| gender | пол |
| study_time_hours | количество часов подготовки |
| attendance_percent | процент посещаемости |
| sleep_hours | среднее количество часов сна |
| parental_education | образование родителей |
| internet_access | наличие доступа к интернету |
| extracurricular_activities | участие во внеучебной деятельности |
| part_time_job | наличие подработки |
| previous_grade | предыдущая оценка |
| final_exam_score | итоговый балл экзамена |
| final_grade | итоговая оценка |

## Результаты анализа

В ходе проекта были исследованы факторы, влияющие на итоговую успеваемость студентов.

### Основные выводы

- Средний итоговый балл экзамена составил **83.54**.
- Средняя посещаемость — **85.09%**.
- Среднее время подготовки — **3.57 часа**.

### Влияние посещаемости

Студенты с высокой посещаемостью (90% и выше) показали средний результат **86.41**, тогда как при низкой посещаемости средний балл составил **78.29**.

Разница между группами — **8.12 балла**.

### Влияние времени подготовки

Студенты, которые занимались **5 часов и более**, получили средний балл **92.14**.

Студенты, занимавшиеся менее **3 часов**, получили **77.21**.

Разница составила **14.93 балла**, что делает время подготовки одним из самых значимых факторов.

### Доступ к интернету

Средний балл студентов с доступом к интернету оказался выше (**84.28**) по сравнению со студентами без доступа (**79.26**).

Разница составила **5.02 балла**.

### Образование родителей

Связь между уровнем образования родителей и итоговым баллом оказалась слабой.

Разница между максимальным и минимальным средним баллом составила всего **1.83 балла**.

### Подработка

Студенты без подработки показали немного более высокий средний результат (**84.58**) по сравнению со студентами, совмещающими учебу и работу (**81.29**).

Разница составила **3.29 балла**.

### Дополнительный анализ

В проекте также использовались:

- CTE (Common Table Expressions);
- оконные функции (Window Functions);
- ранжирование студентов;
- поиск студентов выше среднего уровня;
- определение группы риска;
- поиск лучших студентов в каждой группе образования родителей.
